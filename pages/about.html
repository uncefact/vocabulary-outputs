UN/CEFACT Web Vocabulary

https://vocabulary.uncefact.org/

About

The UN/CEFACT Web Vocabulary is a linked data representation of the UN/CEFACT Buy-Ship-Pay Reference Data Model. The BSP RDM generalizes concepts of Multi-Modal Transport Reference Data Model (MMT-RDM) and the Supply Chain Reference Data Model (SCRDM), which can be applied by country and regional administrations and industries.

Version

The vocabulary reflects BSP version 20A.

Contexts

JSON-LD Context files are available here:

https://vocabulary.uncefact.org/unece-context.jsonld

https://vocabulary.uncefact.org/unlocode-vocab-context.jsonld

Content Negotiation

Machine readable JSON-LD term definitions are accessible using HTTP content negotiation:

curl https://vocabulary.uncefact.org/ -H 'Accept: application/ld+json'

curl https://vocabulary.uncefact.org/unlocode# -H 'Accept: application/ld+json'
